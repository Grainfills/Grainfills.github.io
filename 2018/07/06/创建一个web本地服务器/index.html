创建一个web本地服务器

Jul 6, 2018 | 服务器 | 159 字 | 1 分钟
条评论

快速创建node.js本地web服务器，用于调试

初始条件

  1. 全局安装 node
  2. 全局安装 http-server

启动使用

  1. mkdir http-server
  2. cd http-server
  3. http-server

运行结果为 http-server -a 192.168.xxx.xxx -p 7070

参数以及设置

http-server 参数说明

  • -p 端口号 （默认8080）
  • -a IP地址
  • -d 显示目录列表（默认ture）
  • -s 日志信息输出
  • -o 开始服务后打开浏览器
  • -u 或—utc在日志消息中使用UTC时间格式。

我的使用

文件夹放在收藏栏，vue编译结束拖到文件夹就可以跑起来查看效果
